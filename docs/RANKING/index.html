評価基準
選ばれし四人のメロンソムリエ(笑)が、
この世のメロン味を冠する食べ物を
以下の四つの観点で評価する。
・単純な美味しさ
・見た目の近さ
・味の近さ
・匂いの近さ
マイスターの中には
メロン嫌いも在籍しており、
非常に厳しい審査となっている。
ときに厳しい感想も見られるが、
彼らによる至極の判定は
きっとあなたの甜瓜人生の
糧になるはず。
是非あなたご自身も
ソムリエになったつもりで
ご覧いただきたい。
甜瓜の未来に幸あれ。
メロンメーター
一位　メロン
総評：これをメロンと言わずしては基準が成り立たない。色、フォルム、香り、味、全てにおいて満点メロンである。まぁ当たり前である。
メロンメーター
二位　コロロ
総評：まさか二番目に近しいのがグミとは、誰が想像できただろうか、いやできまい。しかしその匂いから色に至るまで、以外にも優れたバランスを有している。隠れたメロ二スト、おめでとう、コ◯ロ。
メロンメーター
三位　ゼリー
総評：メロンの表現としては、色や匂いといった部分で、少しばかり誇張が過ぎるようだ。昆虫ゼリーのようだともあったが、真相やいかに。
メロンメーター
四位　スイカ
総評：刻まれし漆黒の文身は、そのものの持つ味わい深さとはまた違う魅力を放つ。丸みを帯び、ずっしりと構える姿は、まさに圧巻。思ってるより匂いが近いらしいよ、この野菜。
メロンメーター
五位　アメ
総評：ギリ、まだメロンとしての体裁を保てている。丸いし、オレンジだし、おいしいし。アメの硬いとことか、安いメロンの食感に近いんじゃね？
（そろそろ庇いきれなくなってきたな,,）
メロンメーター
六位　キャラメル
総評：いや、まだいける。溝とかあるし、匂いまじびっくりするくらい一切ないけど、味遠くないらしいし。
おいパイナップルっつったの誰だ。
メロンメーター
七位　メロンパン緑
総評：いいね、ここで若干巻き返してきたやん。緑だし。しかも中身がオレンジなのもね、ギミックとしては共通項だし。
（擁護派消えたな）
メロンメーター
八位　メロンパン黄
総評：すべての元凶。お前がメロンを名乗らなければこんなことには。なぜだ、なぜそなたはメロンを語る。どっちかって言えばカメやん。色もちゃうやん。まぁ美味いんだが。
メロンメーター
九位　ソーダ
総評：なぜかランキングに参入。基準は不明。おそらく選定委員会にクリーム好きがいたのだろう。クリームがメロンを引き立てているのではという感想は非常に興味深いものであった。
メロンメーター
十位　FANTA
総評：元凶セカンド。何味かと問われれば迷うことなくメロンと答えるが、じゃあメロンですかと聞かれりゃ無理がある、腹立たしいくらいに。てかよく見たら何これ、めっちゃ毒々しい色。いやこっわ。何味これ、ねぇ何味？怖いよ。
